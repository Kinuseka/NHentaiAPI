# NHentai API
A python script that collects data from NHentai.net. 

**Goals**
```
>Collect data from NHentai
>Automatically save the metadata(tags)
>Download pages concurrently
>API Template (For possible other websites to support)
```

**Libraries used**
```
>bs4 
>Httpx
>Anyio (Trio Backend)
>Trio
>pyyaml
```
